# Sales Data Analysis

## Project Overview

This project analyzes a Superstore Sales dataset using Python and Pandas.

## Dataset Information

* Rows: 9800
* Columns: 18

## Tools Used

* Python
* Pandas
* Google Colab

## Analysis Performed

* Data Exploration
* Customer Analysis
* Category Analysis
* Sub-Category Analysis
* Regional Analysis
* Date Analysis
* Missing Value Analysis

## Key Findings

* 2018 generated the highest sales.
* November generated the highest monthly sales.
* Technology was the highest revenue category.
* Phones were the highest revenue sub-category.
* California generated the highest state sales.
